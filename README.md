Wedding Name Tags Rental Website

=========================================================================

A modern, elegant, and mobile-responsive website for wedding name tag rentals in Malawi.
Designed to present premium wedding stationery, rental terms, and enquiry options in a clean, trustworthy format.

=========================================================================

Live Purpose

=========================================================================

This website serves as a digital catalogue and enquiry platform for customers interested in renting wedding name tags.
It focuses on elegance, clarity, and ease of contact rather than direct online payments.
